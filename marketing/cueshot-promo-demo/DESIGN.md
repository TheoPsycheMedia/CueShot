# CueShot Promo Demo Visual Identity

## Style Prompt

Create a restrained premium macOS product demo for CueShot: graphite glass, pearl typography, precise optical-reticle accents, and privacy-safe coded UI as the hero. The video should feel like a compact technical instrument demo, not a SaaS explainer deck. Motion is confident and quiet: focus pulls, measured pushes, reticle pulses, and clean interface zooms.

## Colors

- `#111516` canvas graphite
- `#1D2221` raised panel graphite
- `#F3EFE6` pearl primary text
- `#A9AEAA` muted secondary text
- `#38D98B` reticle green accent
- `#2D3432` bevel and divider stroke

## Typography

- Headline/body: `Space Grotesk`, 360-850, precise technical UI voice
- Metadata and dimensions: `SF Mono`, tabular numbers

## Motion

- Medium-premium pacing: 0.35-0.7 second entrances.
- Use focus-pull and push transitions; avoid jump cuts.
- Reticle accent may pulse slowly, but no infinite repeat. Calculate finite repeats.
- Coded UI panels should move like physical glass: scale, y, blur, and opacity only.
- Highlights must be attached to the coded UI elements they describe, not manually offset over screenshots.

## What NOT To Do

- No purple/blue gradients, SaaS blobs, bokeh, or decorative orbs.
- No busy dashboard montage; the capture loop is the story.
- No tiny captions that only work on a webpage.
- No real desktop screenshots or personal source material.
- Do not use public claims about official Codex APIs.
