# CueShot v0.1.8

CueShot v0.1.8 is a presentation and product-design release. It preserves the capture engine, permissions, history, clipboard-first behavior, shortcuts, and Codex handoff semantics while refreshing the app into a calmer, more premium Mac capture instrument.

## Highlights

- Reworked the main window from a diagnostic dashboard into a capture review workspace.
- Added a hybrid old/new layout with a polished capture lens and a compact widget rail.
- Restored the older floating control menu shape while keeping the newer branded glass and theme colors.
- Added CueShot brand primitives: brand mark, lockup, lens aura, branded sheen, and Liquid Glass-aware surface helpers.
- Added theme-driven widget gradients so Optic, Aurora, Moss, and Cinder visibly affect the app surface.
- Moved metadata, file details, permissions, and troubleshooting behind deliberate Details/Troubleshooting disclosure.
- Rebuilt Settings into intent-based tabs: General, Capture, Files & History, Shortcuts, and Privacy.
- Improved capture copy: user-facing language now says Capture, Copy Again, Copy Text, Show in Finder, and Paste command sent, check Codex.
- Preserved clipboard-first truth: CueShot copies first, and optional visible Paste never claims Codex attached the image unless verified.

## Compatibility and behavior

- Capture modes remain available: Element, Selection, Area, Window, Screen, and OCR/Text.
- Existing shortcuts, history storage, PNG clipboard behavior, drag preview, save, reveal, OCR copy, and optional Codex handoff remain intact.
- macOS 26 Liquid Glass use remains behind availability/compiler checks with material fallbacks for macOS 14 and later.
