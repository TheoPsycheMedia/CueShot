# CueShot v0.1.6

Release date: 2026-06-22

## Highlights

- Adds OCR mode for estimated-region text capture, OCR preview, and copyable recognized text.
- Restores the visible Codex paste path using a clipboard-first handoff, Codex activation, a lower-composer focus click, and System Events `Edit > Paste`.
- Adds Automation/System Events as a first-class macOS permission in onboarding, Settings, diagnostics, and the permission helper script.
- Improves element selection behavior so resized capture targets move with the mouse without resetting size.
- Keeps CueShot overlays out of captured pixels by hiding capture UI before freezing the target.
- Improves drag payloads from the capture preview by publishing file URL, URL name, path text, and PNG data.

## Permission Notes

CueShot now reports all three relevant macOS privacy surfaces:

- Accessibility for element targeting, event monitoring, and local focus/click automation.
- Screen Recording for visible pixel capture.
- Automation/System Events for the optional visible paste handoff into Codex.

Automation cannot be silently pre-granted by CueShot. macOS owns that grant, but CueShot now declares the correct purpose string, shows the current status, opens the Automation privacy pane, and includes it in `./script/build_and_run.sh --diagnose`.
