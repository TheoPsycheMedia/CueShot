# Support

For bugs, use GitHub Issues with the bug report template.

For feature ideas, use the feature request template and describe the workflow you want to improve.

For vulnerabilities or privacy-sensitive capture behavior, follow [SECURITY.md](SECURITY.md) instead of opening a public issue.

Helpful details for support requests:

- macOS version.
- CueShot commit or release.
- Capture mode used.
- Whether Accessibility and Screen Recording permissions are granted.
- A privacy-safe screenshot or short recording when the issue is visual.
